React Plx

Scroll down to get things moving!

Plx - React parallax component, lightweight and powerful. Supports animating CSS properties that accept numerical values (e.g. opacity, transform, height...). Also supports colors. It has built-in easing functions, or you can pass your own function or custom cubic bezier.

Detailed documentation available on GitHub.
